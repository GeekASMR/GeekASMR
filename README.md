### What I work on

- 🔊 **Windows audio internals** — kernel WDM drivers, virtual audio endpoints, low-latency IPC, VST3 hosting. Maintainer of [`WDM2VST-Ultra`](https://github.com/GeekASMR/WDM2VST-Ultra) (system-audio ↔ DAW bridge), [`UMC-Ultra-drivers`](https://github.com/GeekASMR/UMC-Ultra-drivers), [`ASIO-Ultra-drivers`](https://github.com/GeekASMR/ASIO-Ultra-drivers).
- 🎚 **Hackintosh OpenCore** — Z370/Z390 Phantom ITX configs and friends. Repos under `*-OpenCore-Hackintosh`.
- 🎛 **DAW localization & tooling** — [LUNA Simplified Chinese patch](https://github.com/GeekASMR/UA-LUNA-Simplified-Chinese-Patch), [`UAD-Plugin-Manager`](https://github.com/GeekASMR/UAD-Plugin-Manager).
- 🪟 Random Windows quality-of-life tools — [`window-drawer`](https://github.com/GeekASMR/window-drawer).

### Currently helping

- 🎯 Driving WDM2VST Ultra toward Microsoft WHQL signing so anti-cheat systems stop blocking it. [Donate / sponsor →](https://ultra.asmrtop.cn/donate/)
- 🎧 Iterating on a network audio bridge for collaborative ASMR / monitoring sessions.
